# NIGIRI / TSUMAMI 専用設定ツール (Mouse Config Tool)

## 概要
本ソフトウェアは、FPS等において「目を閉じていても狙い通りに動かせる」ことを目指して開発されたカスタム有線マウス**「NIGIRI」「TSUMAMI」**専用の設定ツールです。

デバイスはPCに接続するだけでも動作しますが、本ツールを使用して詳細なパラメータを設定し、**マウス本体（オンボードメモリ）へ書き込む**ことで、そのポテンシャルを最大限に引き出すことができます。設定内容は本体に記憶されるため、ゲームのプレイ中にソフトウェアを常駐させる必要はありません。

## 対応ハードウェア仕様
持ち方の安定性をエイムの土台とし、目的（トラッキングの安定か、フリックの速さか）に合わせて極限まで無駄を削ぎ落とした専用設計となっています。

* **NIGIRI (リバース持ち専用設計)**
  * 手のひらと指の腹を使って面で固定し、精密なトラッキングを安定させます。
  * ホイールとクリックを同時に操作できるよう、親指側に右クリック、中指でホイールを操作できる独自のボタン配置を採用しています。
  * 後部パーツは手の大きさに合わせて交換可能です（重量約52g）。
* **TSUMAMI (つまみ持ち専用設計)**
  * 指先だけで保持し、素早いエイム（フリック）を実現します。
  * 薬指と小指部分は波形状でフィット感を高め、親指部には滑り止めの専用シリコンシートを配置。さらに、指の底の板を利用して物理的なブレーキをかけられる構造です（重量約45g）。

## 独自の設定機能
人間の腕は直線ではなく、手首や肘を支点とした円弧を描いて動くため、通常の直線的なセンサー認識では感覚と実際のカーソル移動にズレが生じます。これを補正し「感覚を一致させる」ため、以下の独自機能を搭載しています。

* **XY軸 独立DPI・センサー角度設定**
  上下（Y軸）と左右（X軸）のDPI、およびセンサー角度を個別に調整可能。個人の骨格や可動域に由来する感覚のズレを補正し、実際の動きを画面へ正確に反映させます。
* **独自機能「Q2フィルタ」**
  マウスを「遅く動かした（減速した）時」にのみDPIを自動で下げるフィルタリング機能です。これにより、「大きく素早く振り向くための高DPI」と、「的の近くで精密に止めるための低DPI」を両立させ、速度と精度のジレンマを解決します。

## インストールと使用方法
1. 本リポジトリの [Releases](https://github.com/Mouse-Laboratory/NIGIRI-TSUMAMI_Software/releases/latest) より、最新のインストーラーをダウンロードしてください。
2. ダウンロードしたインストーラーを実行し、画面の指示に従ってインストールを完了させてください。
   * ※ダウンロード時や実行時に「WindowsによってPCが保護されました」という警告画面が表示される場合があります。その際は「詳細情報」をクリックし、「実行」を選択してください。
3. インストールされたソフトウェアを起動します。
4. マウスを接続した状態で、各種パラメータ（DPI、角度、Q2フィルタの効き具合など）をご自身の感覚に合わせて設定してください。設定値はマウス本体へ書き込まれます。
